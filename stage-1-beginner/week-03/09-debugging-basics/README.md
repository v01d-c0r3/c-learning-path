# Compiling with warnings & first debugger session

Use compiler warnings and gdb/lldb to find and fix bugs.

## Learning goals

- Compile with -Wall -Wextra -Werror -g
- Set breakpoints and step through code in gdb
- Inspect variable values at runtime
- Understand a basic backtrace
