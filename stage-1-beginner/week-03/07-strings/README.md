# Strings as null-terminated char arrays

Understand how C represents text and use the standard string library safely.

## Learning goals

- Declare string literals and char arrays
- Use strlen, strcpy, strcat, strcmp, strstr
- Avoid buffer overflows with bounded alternatives
- Iterate character by character
