# Hello, World & the compilation pipeline

Understand how a .c file becomes an executable via preprocessing, compilation, assembly and linking.

## Learning goals

- Explain the four stages of compilation
- Use gcc with -Wall -Wextra -Werror
- Understand #include and the preprocessor
- Run your first compiled binary
