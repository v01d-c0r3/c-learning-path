# Operators, expressions & precedence

Master arithmetic, bitwise, logical and assignment operators, and understand evaluation order.

## Learning goals

- Use +, -, *, /, % correctly
- Apply bitwise &, |, ^, ~, <<, >>
- Understand short-circuit evaluation with && and ||
- Predict results of mixed-type expressions
