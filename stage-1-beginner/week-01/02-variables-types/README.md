# Variables, primitive types & sizes

Learn C's built-in types, their sizes on common platforms, and how to print them correctly.

## Learning goals

- Declare int, float, double, char, unsigned int
- Use sizeof() to inspect type sizes
- Match types to correct printf format specifiers
- Understand signed vs unsigned
