# TCP/UDP socket programming

Write a minimal TCP echo server and client from scratch.

## Learning goals

- socket / bind / listen / accept / connect
- send / recv for TCP, sendto / recvfrom for UDP
- Set SO_REUSEADDR
- Handle partial reads/writes

## Exercises

2. Complete the TODO sections in `echo_server.c`.
3. Compile with: `gcc -Wall -Wextra -Werror -o out echo_server.c && ./out`
