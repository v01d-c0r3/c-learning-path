# Pointer arithmetic & array decay

Navigate arrays using pointer arithmetic and understand the array-pointer relationship.

## Learning goals

- Increment/decrement pointers to walk arrays
- Understand that array names decay to pointers
- Compare pointer subtraction to get distances
- Use pointers vs indices interchangeably
