# Dynamic memory: malloc / calloc / realloc / free

Allocate and manage heap memory safely.

## Learning goals

- Allocate arrays of runtime size with malloc
- Zero-initialise with calloc
- Grow allocations with realloc
- Free every allocation and avoid double-free
