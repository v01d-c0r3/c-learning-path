# Multi-file projects, headers & Makefiles

Split a C project across multiple files and build it with make.

## Learning goals

- Write header guards (#ifndef / #define / #endif)
- Declare vs define — what goes in .h vs .c
- Use extern for shared globals
- Write a Makefile with all, clean, and object targets

## Files

| File | Purpose |
|------|---------|
| `main.c` | Starter file — fill in the exercises below |

## Exercises

1. Read the learning goals above.
2. Complete the TODO sections in `main.c`.
3. Compile with: `gcc -Wall -Wextra -Werror -o out main.c && ./out`
4. Tick this topic off your checklist once all exercises pass.
