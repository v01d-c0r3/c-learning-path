# Function pointers & callbacks

Store and invoke functions through pointers to enable flexible, generic code.

## Learning goals

- Declare function pointer variables
- Assign and call functions through pointers
- Build a dispatch table (array of function pointers)
- Write a generic sort with a comparator callback
