# File I/O: fopen, fread, fwrite, fclose

Read from and write to files in both text and binary mode.

## Learning goals

- Open files safely and check for NULL
- Read lines with fgets
- Write formatted output with fprintf
- Read/write binary structs with fread/fwrite
