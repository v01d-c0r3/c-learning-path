# Singly & doubly linked lists

Build generic linked list implementations from scratch.

## Learning goals

- Implement insert, delete, search for singly linked list
- Implement a doubly linked list with prev pointers
- Reverse a singly linked list in-place
- Detect a cycle using Floyd's algorithm

## Files

| File | Purpose |

## Exercises

1. Read the learning goals above.
2. Complete the TODO sections in `linked_list.c`.
3. Compile with: `gcc -Wall -Wextra -Werror -o out linked_list.c && ./out`
4. Tick this topic off your checklist once all exercises pass.
