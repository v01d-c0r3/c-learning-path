# POSIX basics: open/read/write, fork/exec/wait

Interact with the operating system directly using POSIX system calls.

## Learning goals

- Open/read/write files with POSIX file descriptors
- Fork a child process
- Replace a process image with exec
- Wait for children and harvest exit codes
