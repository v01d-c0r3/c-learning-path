# Quicksort, mergesort & heapsort

Implement the three canonical O(n log n) comparison sorts.

## Learning goals

- In-place quicksort with Lomuto partition
- Mergesort with auxiliary array
- Heapsort using a max-heap
- Benchmark all three on 100k random ints
